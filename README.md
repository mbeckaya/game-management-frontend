# game-management-frontend

This is a **Vue.js** frontend application for managing Game resources.
It is built with **Vue 3**, **Vue Router**, **TypeScript**, and styled with **Tailwind CSS** and **DaisyUI**.
Currently under development.

## Installation

Copy the `.env.example` file to `.env` and fill in the required values.
```bash
cp .env.example .env
```

Install dependencies and start the app.
```bash
npm install

npm run dev
```
